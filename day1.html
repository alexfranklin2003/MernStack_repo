Web Development Basics:

Today, I learned the foundational concepts of web development. These include understanding how web applications work, the importance of protocols, and the role of IP addresses in communication between devices. Here's a brief summary:

5 Basic Terminologys--

1. Protocols: Protocols are rules that govern communication between applications. I learned about the **client-server architecture** and how data flows between the client (browser) and the server (backend).


2. IP Address:

IP addresses are unique identifiers for devices on a network. I learned about: - **Private IPs** (used within local networks). - **Public IPs** (used to access devices over the internet). Example: Swiggy uses public IP addresses to serve its website. Challenges: Memorizing IP addresses is difficult, which is why we use domain names.


3. Domain Name System (DNS):

The system that maps domain names (like swiggy.com) to their IP addresses.


4. Domain Registrars:

Companies that allow us to register domain names for websites.


5. URLs:

Uniform Resource Locators are the complete web addresses we use to access resources on the internet.

My First HTML File

I practiced creating an HTML file using VS Code. - Commands used: `cd` to navigate directories, `mkdir` to create folders. - HTML files are saved with the `.html` extension. - Shortcut for HTML5 structure in VS Code: `html:5`.

Heading Tags and Paragraphs:

I practiced using heading tags (h1 to h6) and paragraph tags ('p') to structure content. These tags help create a clear hierarchy for the webpage.
